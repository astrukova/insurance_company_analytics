# Единый обзорный дашборд для страховой компании 

## Цель проекта
Единый обзорный дашборд для руководителей всех подразделений страховой компании с базовой статистикой о клиентах. 
Цели использования дашборда: регулярное планирование на разные периоды (месяц, квартал или год), изучение целевой аудиторию для определения перспективных зон развития бизнеса. 

Информация в отчёте разделена на два основных блока:
- Источники доходов компании.
- Портрет клиента: анализ базовых статистических показателей.

## Используемые технологии
Yandex DataLens, SQL. 

## Используемые данные
Данные вымышленной страховой компании, используются в качестве примера.

Описание полей данных:
- `age` — возраст клиента, в годах.
- `sex` — пол клиента.
- `bmi` — ИМТ, индекс массы тела.
- `children` — количество детей у клиента.
- `smoker` — курит (1) или нет (0).
- `region` — один из четырёх районов проживания клиента.
- `charges` — страховые взносы клиента.
